# 🛍️ E-commerce платформа "Scent Vault"

> Проект интернет-магазина селективной парфюмерии и распива (отливантов), разработанный в рамках учебного семинара ИТ-СИП-23-24-ФР-2.

* [🛠️ Техническое задание и User Flow (terms.md)](terms.md) — Функциональные и технические требования к сайту, архитектурный стек и сценарии поведения пользователей.

* [📊 Оценка проекта и конкуренты (review.md)](review.md) — Критерии оценки успешности продукта и сравнительный анализ с лидерами рынка.

## 💻 Технологический стек (Tech Stack)

* **Frontend:** HTML5, CSS3 (методология БЭМ для переиспользуемости компонентов), JavaScript (ES6+).
* **Дизайн и прототипирование:** Figma.
* **Инструменты оптимизации:** Google PageSpeed Insights, TinyPNG / WebP Converter, Chrome DevTools.
* **Контроль версий:** Git / GitHub.
